# A Study in Steampunk: Choice by Gaslight (蒸汽朋克研究：煤气灯下的抉择) 简中汉化项目

## 📖 项目简介
本项目为 Hosted Games 出品的互动小说《A Study in Steampunk: Choice by Gaslight》的中文本地化项目。

该作由 Heather Albano 编写，总文本量约 27.7 万字。游戏的所有剧情分支与数值判定均由 ChoiceScript 代码驱动。本项目通过自动化工具配合 AI 技术实现了全文本覆盖，并建立此仓库以支持后续的人工翻译润色。

- **当前版本**：v0.1 (AI 初版)
- **翻译技术**：基于 Gemini 3 pro 模型进行全文本自动化处理。
- **项目状态**：已完成全文本 AI 翻译，后续将考虑进行人工汉化。

## 🛠️ 技术指南与工作流
本项目基于 Electron 架构。游戏在运行时并不直接读取 .txt 文件，而是读取同名的 .txt.json 字典文件，因此文本修改后必须经过同步步骤。

### 1. 环境准备
使用本项目工具前，请确保系统中已安装 Node.js。

### 2. 解包资源 (Unpack)

1. **准备原始文件**：从正版游戏的安装目录（`resources` 文件夹）中，将以下两个内容一并放入本项目根目录：
   - **`app.asar`**：核心资源包。
   - **`app.asar.unpacked`** 文件夹：包含游戏运行所需的二进制依赖（如 `.node`, `.dll` 等文件）。
   
   > **注意**：这两个内容必须成对存在。若缺失 `.unpacked` 文件夹，会导致解包不完整或封包后的游戏无法正常启动。

2. **执行解包**：运行 `unpack.bat`，脚本会将资源解压至 `unpacked/` 文件夹。

### 3. 文本修改与资源替换逻辑

为了确保补丁的完整性，协作时请根据文件类型前往对应路径进行修改：

#### 📝 剧情文本 (Scenes)
- **存放路径**：`unpacked/deploy/scenes/`
- **修改准则**：
  - 请对照 `/source_en` 中的原文进行翻译。
  - **严禁修改**任何以 `*` 开头的指令行（如 `*if`, `*choice`, `*set`, `*label`）。
  - **缩进规范**：ChoiceScript 极其依赖缩进。请务必保持行首空格数量与原文件完全一致，否则会导致游戏报错。

#### 💻 UI 界面与系统脚本 (UI)
- **存放路径**：`unpacked/deploy/` 
- **核心文件**：重点关注 `index.html`, `ui.js`, `scene.js`, `mygame.js`, `util.js`。
- **修改准则**：
  - 仅限修改双引号内的文本字符串（如菜单标题、按钮标签）。
  - **禁止改动函数逻辑与变量名**，否则可能导致 UI 无法渲染或存档功能失效。
  - 确保文件保存编码为 **UTF-8 (无 BOM)**。

#### 🖼️ 图片资源 (Images)
- **存放路径**：位于 `unpacked/deploy/`目录下。
- **替换方法**：
  - 参考 `/images_en` 确认原图位置。
  - 将 `/images_cn` 中汉化好的图片（保持原文件名与格式）覆盖至 `unpacked` 对应的原始路径。
  - 建议保留原图的尺寸与分辨率，以防止 UI 布局错位。

### 4. 同步索引 (Convert) - 关键步骤
修改完 .txt 文本后，必须更新对应的 JSON 字典：
- 运行 convert_all.bat。
- 该工具会通过 convert.js 扫描文本中的 *label 标签位置并生成全小写索引，确保跳转逻辑正确。

### 5. 重新封包 (Pack)

1. **执行封包**：运行 `pack.bat`。
2. **自动化处理逻辑**：
   - **自动备份**：脚本会首先将现有的 `app.asar` 备份为 `app.asar.bak`，以防封包失败导致资源丢失。
   - **依赖分离**：脚本使用专业命令将 `.node`、`.dll`、`.lib` 等二进制文件排除在压缩包外，并自动更新/同步至 `app.asar.unpacked` 文件夹中。
3. **运行建议**：
   - 封包完成后，必须确保生成的 **`app.asar`** 与 **`app.asar.unpacked`** 文件夹同时被复制回游戏的安装目录。
   - 缺少其中任何一项均会导致游戏启动时弹出“File not found”错误或黑屏。

- **逻辑点**：在该文件中搜索 *scene_list 指令，其下方按顺序罗列的文件名即为游戏剧情的实际加载与衔接逻辑。

## 🎮 玩家安装说明
若您仅需安装已发布的汉化补丁：

1. **定位目录**：在 Steam 库中右键点击游戏 -> 管理 -> 浏览本地文件。进入 resources 文件夹。
2. **备份原件**：将原始 app.asar 重命名为 app.asar.bak。
3. **覆盖补丁**：将下载的汉化版 app.asar 放入该目录。
4. **运行要求**：确保 app.asar 与解包时生成的 app.asar.unpacked 文件夹处于同一级目录下，否则游戏将无法启动。

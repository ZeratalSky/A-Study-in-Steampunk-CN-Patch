# A Study in Steampunk: Choice by Gaslight (煤气灯下的抉择) 汉化补丁

## 项目简介
本项目为 Hosted Games 出品的蒸汽朋克题材互动小说《A Study in Steampunk: Choice by Gaslight》（中文译名：蒸汽朋克研究：煤气灯下的抉择）的中文本地化补丁。

该作由 Heather Albano 编写，包含约 27.7 万字的庞大文本量。由于原著完全基于文字驱动且逻辑分支极其丰富，本项目通过技术手段实现了全文本覆盖，旨在优化中文玩家的阅读与沉浸体验。

**当前版本：v0.1 (AI 汉化版)**
- **翻译技术**：采用 Gemini 大语言模型进行自动化翻译。
- **补丁形式**：基于 Electron 框架封装的 `app.asar` 资源包。
- **适配版本**：Steam 最新发行版。

## 安装说明

本补丁通过替换核心资源包实现汉化。请务必严格按照以下步骤进行操作，以确保补丁生效并能够随时还原：

### 1. 定位游戏目录
在 Steam 库中右键点击游戏 -> **管理** -> **浏览本地文件**。
或者直接导航至以下默认安装路径：
`C:\Program Files (x86)\Steam\steamapps\common\StudyInSteampunkChoiceByGaslight\resources\`

### 2. 备份原始文件
在 `resources` 文件夹中找到原始的 **`app.asar`** 文件。
将其重命名为 **`app.asar.bak`**。
> **注意**：备份文件对于后续还原或版本更新至关重要，请勿直接删除。

### 3. 应用汉化补丁
将从本仓库下载的汉化版 **`app.asar`** 文件复制并粘贴到上述 `resources` 文件夹中。

### 4. 启动游戏
通过 Steam 正常启动游戏。若安装成功，游戏主界面及文本内容将显示为中文。
